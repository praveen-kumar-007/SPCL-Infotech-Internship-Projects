THE GOLDEN SPOON RESTAURANT
===========================

A modern, elegant restaurant website built with React.
This project showcases a stylish landing page for The Golden Spoon Restaurant,
featuring sections like Home, About, Menu, Testimonials, and Contact,
with a clean design and smooth navigation.

------------------------------------------------------------
FEATURES
------------------------------------------------------------
- Home Page – Hero section with tagline and call-to-action
- About Section – Story of the restaurant and its culinary journey
- Menu Section – Starters, main courses, and today’s specials
- Testimonials – Customer reviews and experiences
- Contact Section – Address, phone, email, and working hours
- Responsive Design – Optimized for desktop and mobile

------------------------------------------------------------
INSTALLATION
------------------------------------------------------------
Clone the repository and install dependencies:

git clone https://github.com/praveen-kumar-007/SPCL-Infotech-Internship-Tasks.git
cd the-golden-spoon-restaurant
npm install make

------------------------------------------------------------
USAGE
------------------------------------------------------------
Start the development server:
npm start

Build for production:
npm run build
